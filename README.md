# Self-Refining Engine (SRE)

> AI 데이터 개발자의 기술 블로그(TIL) & Life OS — 픽셀아트 + 코지 게이미피케이션 대시보드

🌐 **Live Site**: [hyunNus.github.io](https://hyunNus.github.io)

## 주요 기능

- **홈 대시보드**: 픽셀아트 캐릭터, XP 레벨 시스템, 스킬 트리, 활동 히트맵
- **TIL**: DEV / PAPER / STACK 카테고리, 검색/정렬, XP 시스템
- **Life OS 독서 트래커**: 상태 관리, 장르 필터, 월별 차트, 독서 목표

## 기술 스택

HTML5 / CSS3 / Vanilla JS / LocalStorage / Canvas API / GitHub Pages
